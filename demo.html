Basic Usage

The simplest way to use the carousel with default settings:

<div class="product-carousel-container" id="basicCarousel" data-carousel>
    <div class="product-carousel">
        <!-- Slides go here -->
    </div>
</div>

Disable Auto-Scroll

You can disable auto-scrolling with the data-auto-scroll attribute:

<div class="product-carousel-container" id="noAutoScrollCarousel" data-carousel data-auto-scroll="false">
    <div class="product-carousel">
        <!-- Slides go here -->
    </div>
</div>

Custom Interval

You can set a custom auto-scroll interval (in milliseconds):

<div class="product-carousel-container" id="fastCarousel" data-carousel data-interval="1000">
    <div class="product-carousel">
        <!-- Slides go here -->
    </div>
</div>

Advanced Usage

For more advanced customization, you can create a carousel instance manually:

// HTML
<div class="product-carousel-container" id="advancedCarousel">
    <div class="product-carousel">
        <!-- Slides go here -->
    </div>
</div>

// JavaScript
document.addEventListener('DOMContentLoaded', function() {
    const advancedCarousel = new ProductCarousel('advancedCarousel', {
        autoScroll: true,
        autoScrollInterval: 5000,
        breakpoints: {
            large: 1200,
            medium: 800,
            small: 500
        },
        slidesToShow: {
            large: 3,
            medium: 2,
            small: 1
        }
    });
});
